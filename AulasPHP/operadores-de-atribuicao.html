Operadores de Atribuição

Uma atribuição acontece quando queremos colocar algum valor dentro de uma variável, seja ele um número ou string estática, o resultado de uma expressão, o retorno de uma função ou o conteúdo de outra variável.

Lista dos operadores de atribuição

  • Números (int, float):
    • +=: Para atribuir uma variável a ela mesma mais algum valor. Ex: 
      $n = 5;
$n += 1; //Ou $n = $n + 1;
echo "O valor de n é $n";
// O valor de n é 6
    • -=: Para atribuir uma variável a ela mesma menos algum valor. Ex: 
      $n = 2;
$n -= 7; // Ou $n = $n - 7;
echo "O valor de n é $n";
// O valor de n é -5
    • *=: Para atribuir uma variável a ela mesma vezes algum valor. Ex: 
      $n = 8
$n *= -10; // Ou $n = $n * -10;
echo "O valor de n é $n"
// O valor de n é -80
    • /=: Para atribuir uma variável a ela mesma dividida por algum valor. Ex: 
      $n = 1.25;
$n /= 0.5; // Ou $n = $n / 0.5;
echo "O valor de n é $n";
// O valor de n é 2.5
    • %=: Para atribuir uma variável ao resto da divisão dela mesma por algum valor. Ex: 
      $n = 13
$n %= 2; // Ou $n = $n % 2;
echo "O valor de n é $n";
// O valor de n é 1
  • Strings (string):
    • .=: Para atribuir uma variável a ela mesma mais algum valor. Ex: 
      $s = "Olá "
$s .= "Mundo!"; // Ou $s = $s . "Mundo!";
echo "O valor de s é $s";
// O valor de s é Olá Mundo!
