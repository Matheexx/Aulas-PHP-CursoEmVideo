Operadores Aritméticos

Lista dos operadores aritméticos

  • + é o operador de adição
  • - é o operador de subtração
  • * é o operador de multiplicação
  • / é o operador de divisão Real
  • % é o operador de módulo (resto da divisão)

Exemplo de código usando o operador de adição (+)

$n1 = 3;

$n2 = 2;

$s = $n1 + $n2;

echo “A soma entre $n1 e $n2 é igual a $s”;

O código acima vai mostrar na tela a mensagem:

A soma entre 3 e 2 é igual a 5

Ordem de precedência dos operadores aritméticos

Sempre em uma expressão, os operadores que serão executados são, na ordem:

  1. Parênteses ( )
  2. Operadores de multiplicação, divisão e módulo * / %
  3. Operadores de adição e subtração + -

Por exemplo, considerando o código a seguir:

$media = $nota1 + $nota2 / 2;

A média será calculada de maneira errada, já que segundo a ordem de precedência, a divisão será feita antes. O correto seria escrever:

$media = ($nota1 + $nota2) / 2;

Obtendo valores da URL

Vamos analisar a URL abaixo:

http://localhost/meusite/primeirosite.php?a=5&b=2

No link acima, o arquivo primeirosite.php está sendo chamado dentro da pasta meusite do servidor local. Na linha, serão passados dois valores: a valendo 5 e b valendo 2.

Para pegar esses valores no script PHP, use o código:

$valor1 = $_GET["a"];
$valor2 = $_GET["b"];

No código acima, o parâmetro a (5) passado pela URL será armazenado na variável $valor1. De maneira similar, a variável $valor2 vai conter o valor do parâmetro b (2).

Funções matemáticas

  • abs() : Retorna o valor absoluto de um número.
    Ex: abs(-5) = 5
  • pow() : Calcula uma potência.
    Ex: pow(3,2) = 32 = 9
  • sqrt() : Calcula a raiz quadrada de um número.
    Ex: sqrt(25) = 5
  • round() : Arredonda valores.
    Ex: round(3.8) = 4
  • intval() : Trunca um número. Retorna a parte inteira de um valor.
    Ex: intval(8.7) = 8
  • number_format() : Formata um número Real.
    Ex: number_format(3258.754, 2, “,”, “.”) = 3.258,75

Obs: Ainda existem os métodos ceil() e floor() para arredondamentos para cima e para baixo, respectivamente. A função round() vai usar as regras de arredondamento.
